# 🏛️ 李白不同时期背景设计指南

## 📖 文档说明

本文档为科普游戏《李白的一生》提供不同时期的背景设计，包括：
- 青年时期（25-35岁）：游历求仕
- 中年时期（35-50岁）：长安岁月
- 老年时期（50-62岁）：流放与漂泊

每个时期包含：
- 历史背景介绍
- 场景描述
- 建筑风格
- 色彩氛围
- AI生成提示词

### 🎮 游戏场景通道设计

**重要要求**：每个背景必须包含至少2-3个通向其他场景的通道，这是游戏场景设计的关键要素。

**通道类型**：
- **门/拱门**：宫门、拱门、月门等
- **桥**：石桥、木桥等
- **道路/小径**：石板路、山路、小径等
- **传送点**：特殊传送区域（可选）
- **水路**：码头、运河入口等

**通道设计要求**：
1. **清晰可见**：通道应该有明显的视觉标识，方便玩家识别
2. **风格一致**：通道应该与场景风格一致，自然融入场景
3. **位置合理**：通道应该分布在场景的不同位置（左、右、上、下、中央等）
4. **易于识别**：可以使用稍微不同的颜色、光效或标识来突出通道位置

## 🗺️ 三时期区域划分图

### 📐 布局方案

可以用一张图片展示李白三个时期的场景，通过区域划分的方式呈现：

**推荐布局**：
- **横向三分**：左（青年）、中（中年）、右（老年）
- **纵向三分**：上（青年）、中（中年）、下（老年）
- **对角线划分**：左上（青年）、中（中年）、右下（老年）

**最佳方案：横向三分布局** ⭐

**指引设计说明**：
- **青年→中年**：左侧区域右侧边缘有指引（道路、河流、拱门等）指向中央
- **中年→老年**：中央区域右侧边缘有指引（道路、河流、小径等）指向右侧
- **指引形式**：可以是道路延伸、河流流向、建筑排列方向、或明显的视觉标识
- **自然过渡**：指引应该自然融入场景，同时起到引导和通道的作用

## 🌱 青年时期（25-35岁，约725-735年）

### 📚 历史背景

**时期特点**：
- 李白25岁离开四川，开始"仗剑去国，辞亲远游"
- 游历湖北、湖南、江苏、浙江等地
- 求仕未果，但创作了大量优秀诗作
- 生活相对自由，充满理想和抱负

**主要地点**：
- 四川（故乡）
- 湖北（安陆、襄阳）
- 湖南（洞庭湖）
- 江苏（金陵、扬州）
- 浙江（会稽、天台山）

**代表作品**：
- 《静夜思》
- 《望庐山瀑布》
- 《早发白帝城》
